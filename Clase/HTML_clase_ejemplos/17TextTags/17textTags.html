Marcado de texto según su importancia

Utilice b para llamar la atención sobre determinado texto sin indicar un mayor nivel de importancia. Se utiliza para llamar la atención del lector sobre el contenido del elemento, al que de otro modo no se le otorga una importancia especial

Utilice em para marcar el texto que tiene énfasis en el acento.

Utilice i si representa un rango de texto que se desmarca del texto normal por alguna razón, como texto idiomático, términos técnicos, designaciones taxonómicas, etc.

Utilice mark si representa el texto que está marcado o resaltado con fines de referencia o notación debido a la relevancia del pasaje marcado en el contexto de cierre

Utilice strong para una oración que es de gran importancia para toda la página, o para palabras de mayor importancia en comparación con el contenido cercano.

Marcado de texto en cita

Utilizamos

blockquote
para indicar que el texto adjunto es una cita extendida. Para incluir comillas más cortas en línea en lugar de en un bloque separado, utilice el elemento <q> (Cita).

Utilizamos cite para marcar el título de una obra creativa citada como: Un libro. Un trabajo de investigación. Un ensayo. Un poema. Una partitura musical. Una canción. Una obra de teatro o guión de película. Una película. Un programa de televisión. Un juego. Una escultura. Una pintura. Una producción teatral. Una obra de teatro. Una ópera. Un musical. Una exposición. Un informe de caso legal. Un programa informático. Un sitio web. Una página web. Una entrada de blog o comentario. Una publicación o comentario en el foro. Un tweet. Una publicación de Facebook. Una declaración escrita u oral. Etc. Para incluir una referencia a la fuente del material citado que está contenida dentro de un elemento <blockquote> o <q>, use el atributo cite en el elemento.

Utilizamos q para indicar que el texto adjunto es una cita corta en línea. El valor de este atributo es una dirección URL que designa un documento o mensaje de origen para la información citada. Este atributo está destinado a apuntar a información que explica el contexto o la referencia de la cita.

Marcado de texto informático o código fuente

Utilizamos code para mostrar su contenido diseñado de una manera destinada a indicar que el texto es un fragmento corto de código informático.

Utilizamos kbd para representar un intervalo de texto en línea que indica la entrada textual del usuario desde un teclado, entrada de voz o cualquier otro dispositivo de entrada de texto.

Utilizamos 

pre
para representar texto preformateado que debe presentarse exactamente como está escrito en el archivo HTML.

Utilizamos samp para encerrar texto en línea que representa la salida de muestra (o citada) de un programa informático.

Marcado de texto por su interacción con el documento

Utilizamos insertar si representa un intervalo de texto que se ha agregado a un documento.

Utilizamos suprimir si representa un intervalo de texto que se ha eliminado de un documento. Esto se puede usar al representar "control de cambios" o información de diferencias de código fuente.

Utilizamos <details> para crear un widget de divulgación en el que la información solo es visible cuando el widget se cambia a un estado "abierto". Se debe proporcionar un resumen o etiqueta utilizando el elemento <summary>

Utilizamos subíndice Co cuando necesitemos por motivos tipográficos subir o bajar la fuente para su correcto diseño.

Utilizamos supraíndice Logn cuando necesitemos por motivos tipográficos subir o bajar la fuente para su correcto diseño.
